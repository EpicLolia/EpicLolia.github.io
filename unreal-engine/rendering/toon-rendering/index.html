Cpp 侧修改

1. 材质定义

  1. 在EngineTypes.h的枚举EMaterialShadingModel中注册新的光照模型，其会在 Material 的下拉框中显示。注意这里的位置要与后文 HLSL 定义的顺序一致。
  2. 如果想要材质在蓝图中可以通过节点选择，那么还需要在 UMaterialExpressionShadingModel 中添加相应的 Meta
  3. 在MaterialShader.cpp的GetShadingModelString()中增加相应的 Case
  4. 在MaterialShader.cpp的UpdateMaterialShaderCompilingStats更新 Shader Stats

做完以上操作之后，我们就可以在材质中选择我们自定义的光照模型了。

2. Cpp 侧宏导出

HLSL 中是否编译某块代码由宏控制，我们也需要为我们自己的光照模型定义并开启相应的宏

  1. 在ShaderMaterial.h的FShaderMaterialPropertyDefines中添加相关位域，在 HLSL 中宏开关的命名也必须与此处一致。
  2. 在ShaderGenerationUtil.cpp中修改FShaderCompileUtilities::ApplyFetchEnvironment()
  3. 在HLSLMaterialTranslator.cpp的FHLSLMaterialTranslator::GetMaterialEnvironment()添加相应的分支
  4. 在MaterialHLSLEmitter.cpp中也有GetMaterialEnvironment()函数，我们也做同样的操作

3. 材质数据接口
